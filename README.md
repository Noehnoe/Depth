# Depth HTML Workspace

A starter HTML workspace with `index.html` and supporting documentation.

## Included Files
- `index.html` — main landing page
- `styles.css` — basic styling
- `script.js` — simple JavaScript interaction
- `PRD.md` — product requirements document
- `.github/copilot-instructions.md` — workspace instructions checklist

## How to Use
1. Open `index.html` in your browser.
2. Click the demo button to verify JavaScript interaction.
3. Review `PRD.md` for product requirements and goals.
